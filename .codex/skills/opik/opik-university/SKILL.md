# Opik University

Opik University is your home for learning how to get the most out of Opik. Here you'll find a curated collection of video tutorials organized by topic to help you master everything from basic observability to advanced production monitoring.

## When to Use

Use this skill to:

- Point users to specific video lessons in the Opik University series
- Help agents recommend a learning path based on user goals
- Link users to tutorials for Opik features (observability, evaluation, prompt management, testing, monitoring)
- Suggest foundational lessons before advanced topics
